About

4년차 풀스택 개발자로서 AWS 등의 클라우드 및 리눅스 환경에서 웹 서비스를 구축하고 운영해본 경험과 서비스 전반에 대한 높은 이해도를 갖추고 있습니다.

LMS(학습 관리 시스템) 도메인에서 다양한 기능을 개발하며 성능 최적화와 사용자 경험 개선에 기여해왔으며, 새로운 것을 학습하고 적용하는 것을 좋아합니다.

클린 코드와 성능의 효율성, 그리고 지속적인 개선을 추구합니다. "할 수 있을까"보다는 "어떻게 할 수 있을까"를 고민하며 기술적인 도전을 멈추지 않고, 동료들과 함께 문제를 해결해 나가는 과정을 즐기고 있습니다.

Project

실무

  1. 클라우드 LMS 기능 개선

    AI 튜터 연동, WebAuthn 패스키 로그인을 개발했습니다. 안티 조인과 복합 인덱스로 쿼리를 개선해 대시보드 로딩 속도를 90% 단축(20초 → 2초)하고, 비동기 로딩으로 사용성을 개선했습니다.

    • AI 튜터
    • WebAuthn
    • MySQL
    • 성능 최적화

  2. 로그 관리를 위한 ELK 도입

    RDB에 1억 건 이상의 대용량 로그 데이터가 쌓이면서 발생하는 성능 저하를 해결하기 위해, 일부 로그의 처리를 ELK로 대체했습니다. Logstash를 이용해 마이그레이션을 진행하고 스냅샷 백업을 설정했습니다.

    • Elasticsearch
    • Logstash
    • Kibana
    • 마이그레이션

  3. 웹 보안 강화

    ModSecurity와 Fail2ban을 적용해 다중 방어 체계를 구축했습니다.

    • ModSecurity
    • Fail2ban
    • 보안

  4. OTT 클라우드 개발

    2인 팀으로 사용자 페이지 프로토타입을 개발한 뒤, 단독으로 추가 개발 및 안정화 작업을 거쳐 서비스를 정식 런칭했습니다.

    • 프로토타입
    • 단독 개발
    • 서비스 런칭
