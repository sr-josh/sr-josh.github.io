안녕하세요 "개발자" 조승래입니다

창의적이고 혁신적인 웹 솔루션을 만드는 것이 저의 열정입니다

저는

3년의 실무 경력과 그 이상의 경험을 가진 웹 개발자입니다.

사용자 중심의 웹 애플리케이션을 풀스택 개발하며, 새로운 기술을 학습하고 적용하는 것을 좋아합니다.

리눅스와 클라우드 인프라에서부터 프론트엔드와 백엔드 개발까지 웹 서비스 전반에 대한 이해를 바탕으로 다양한 프로젝트에 참여해왔으며

클린 코드와 성능에 있어서의 지속적인 개선을 추구합니다.

"불가능은 없다" 라는 믿음을 가지고 팀원들과 협력하여 복잡한 문제에 도전하고, 풀어나가는 과정을 즐기고 있습니다.

기술 스택

HTML5

시맨틱 마크업과 접근성을 고려한 구조적 웹 개발

CSS3

반응형 디자인, Flexbox, Grid, 애니메이션

JavaScript

ES6+, DOM 조작, 비동기 프로그래밍

React

컴포넌트 기반 개발, Hooks, Context API

Vue.js

Progressive Framework, Vuex, Vue Router

Node.js

서버 사이드 개발, Express.js, RESTful API

Database

MongoDB, MySQL, PostgreSQL

Git

버전 관리, GitHub, 협업 워크플로우

E-Commerce 플랫폼

React와 Node.js를 사용한 풀스택 쇼핑몰 웹사이트. 사용자 인증, 장바구니, 결제 시스템을 구현했습니다.
